# Avaliacao final

> 1°
```
Tendo por base os arquivos e exercícios feitos em aula crie um algoritmo em java 
que solicite a leitura de 10 números inteiros e armazene em um vetor.
Depois ordene este vetor em forma DECRESCENTE, listando em tela os valores ordenados;
```

> 2°
```
Considere uma pilha com 5 posições. Utilizando os arquivos e exercícios feito em aula, 
faça um programa, em java, que solicite ao usuário 5 números inteiros e armazene nesta pilha, 
logo após crie uma segunda pilha de forma que os valores da segunda pilha fique com a posição invertida;
```

> 3°
```
Solicite 10 números inteiros e empilhe, sequencialmente, em uma pilha Crie 2 novas pilhas. 
Finalmente, desempilha todos os elementos da pilha inicial e, 
caso for par empilh em uma das pilhas nova e, caso for ímpar empilhe na outra
```
